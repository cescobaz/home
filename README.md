## Requirements

* npm
* node

```bash
sudo apt install libgpiod2
```

## Usage

Install dependecies

```bash
npm ci
```

Then run server for your configuration.
I have 2 raspberry in my house.

```bash
node index.js SERVER_NAME
```

```bash
node index.js raspi-0-server
```

```bash
node index.js raspi-1-server
```
